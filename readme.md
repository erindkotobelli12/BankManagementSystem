1. Project Topic
    Bank Management System
2. Description of the Project Topic
    Aim of the System
          The aim of the Bank Management System is to create a digital platform that allows banks to manage customer accounts, transactions, and financial records efficiently and securely.
          The system helps automate daily banking operations such as account creation, deposits, withdrawals, transfers, and account monitoring. It reduces manual work, minimizes errors, and improves the speed and reliability of banking services for both employees and customers.
    User Levels
          • Administrator (Admin)
          • Bank Director
          • Bank Employee (Staff)
          • Bank Employee (Teller)
          • Customer
General Requirements of the System
    • User Authentication
    • Secure login system for admins, staff, and customers.
    • Account Management
    • Ability to create, update, and delete bank accounts.
    • Transaction Management
    • Support for deposits, withdrawals, and money transfers.
    • Customer Information Management
    • Store and update customer details securely.    
    • Transaction History
    • Keep records of all financial transactions.
    • Security
    • Protect sensitive data with authentication and authorization.
    • Reporting
• Generate reports for transactions, accounts, and customer activities.
• Database Integration
• Store all information in a structured database for easy access and management.
